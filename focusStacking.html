Focus Stacking Algorithm

This program takes a set of images with different focal depths and converts them into a single high depth of field image. These images are loaded into my program, the images are alligned, and the infocus portions of each layer are calculated. Using the results of this process I was able to rebuild the image with a much greater depth of field than any of the images provided by themselves.
